# Delta Project & Solution – Full Stack Website

## Setup Instructions

### Requirements
- Python 3.8+
- pip

### Installation

```bash
pip install -r requirements.txt
```

### Run

```bash
python app.py
```

Then open: http://localhost:5000

### Admin Access
- URL: Click Login → use admin credentials
- Email: admin@deltaprojecthub.com
- Password: admin123

### Features
- Full product catalog with categories
- Shopping cart (session-based)
- Wishlist
- Order placement with WhatsApp integration
- Customer enquiry form
- Blog
- Newsletter subscription
- Admin dashboard:
  - Manage projects (add/edit/delete)
  - View orders & update status
  - View enquiries
  - Category management
  - Stats dashboard

### Customization
- Add your projects via Admin dashboard
- Update prices, images, descriptions
- All data stored in `delta.db` (SQLite)
